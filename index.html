About Mohr's Circle for Strain

Convention: This plot follows the standard textbook convention (MIT OCW, eFunda):

  • Horizontal axis: Normal strain (ε)
  • Vertical axis: γxy/2 (shear strain, positive upward)
  • Point X: (εx, γxy/2)
  • Point Y: (εy, -γxy/2)

Formulas:

  • εavg = (εx + εy) / 2
  • R = √[((εx - εy)/2)² + (γxy/2)²]
  • ε1 = εavg + R
  • ε2 = εavg - R
  • γmax = 2R
  • tan(2θp) = γxy / (εx - εy)
